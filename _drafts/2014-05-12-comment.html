--- layout: post title: Cisco AnyConnect VPN Client date: '2014-05-12T20:34:00.000-07:00' author: Larry Reid tags: modified_time: '2014-05-12T20:34:02.307-07:00' blogger_id: tag:blogger.com,1999:blog-5778824359157275227.post-9189825253704044133 blogger_orig_url: https://www.blogger.com/comment.g?blogID=5778824359157275227&postID=9189825253704044133 --- Here's how to get the Cisco AnyConnect VPN client working with Ubuntu 10.04 and FireFox.

Cisco AnyConnect needs the Sun Java Runtime Environment (JRE). You can download it from java.com. Put it in /usr/java and expand it (as root).

sudo mkdir /usr/java
sudo chmod a+x jre*.bin
sudo ./jre*.bin
cd /usr/lib/mozilla/plugins
sudo ln -s /usr/java/jre[0-9]*/lib/i386/libnpjp2.so
