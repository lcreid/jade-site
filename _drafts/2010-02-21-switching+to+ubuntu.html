--- layout: post title: Switching to Ubuntu date: '2010-02-21T11:09:00.001-08:00' author: Larry Reid tags: modified_time: '2010-02-21T11:14:18.606-08:00' blogger_id: tag:blogger.com,1999:blog-5778824359157275227.post-2856486275420855295 --- Right now this is just a list of things to consider when someone asks you to install Ubuntu on their computer:
  • Get the make, model, CPU speed, RAM installed and disk size. Check it against the recommended minimums for the current release of Ubuntu, and check the hardware compatability page (especially for laptops)
  • Find out the graphics chip and Google or search Launchpad for issues, especially if it's an older chip
  • Find out if they want to save any work they have on the computer
  • As what they'll be doing with the computer. Are they using any special hardware or software. For example, some of the most popular language learning systems come with hardware and software that only works on Windows (or Mac if you buy the Mac version)
  • Find out what kind of network they'll be hooking to. Check the NIC for compatability (although I haven't had problems with NICs for a couple of years)
  • After installing, do an update. This takes a fair bit of time (overnight)
  • Set up Open Office to default to saving as Microsoft format documents (Word, Excel)
