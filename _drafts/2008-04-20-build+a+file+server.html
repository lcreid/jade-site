--- layout: post title: Build a File Server date: '2008-04-20T08:19:00.000-07:00' author: Larry Reid tags: modified_time: '2008-04-20T08:34:17.382-07:00' blogger_id: tag:blogger.com,1999:blog-5778824359157275227.post-7846317749725899793 --- This post is meant to capture a complete build of an Ubuntu 6.06 general-purpose file server in a way the minimizes the amount of work.
  1. Build the basic VM
  2. Log in to console
  3. Edit /etc/dhcp3/dhclient.conf to send the host name
  4. ifdown and ifup
  5. Install ssh
  6. Log in from a decent shell (e.g. Putty)
  7. Edit /etc/apt/sources.lst to get from universe
  8. sudo apt-get update
  9. sudo apt-get upgrade # this takes some time
  10. Install a whole bunch of things sudo apt-get install ntp-simple snmpd snmp build-essential linux-headers-$(uname -r) bacula-client
